# Sprintz Vercel Website

This is a Vite + React website package for Sprintz Holdings Pte. Ltd.

## Upload to GitHub

Upload these files directly to the root of your GitHub repository:

- package.json
- vite.config.js
- vercel.json
- index.html
- src/
- public/

Do not upload the outer folder itself into the repo root.

## Deploy on Vercel

1. Connect the GitHub repository to Vercel.
2. Let Vercel auto-detect the Vite project.
3. Deploy.
4. Add both `www.sprintz.com.sg` and `sprintz.com.sg` in Vercel domains.
5. Set `www.sprintz.com.sg` as primary and redirect the root domain to it.
